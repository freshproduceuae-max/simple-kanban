# Plans

Per-task implementation plans live here, one file per plan, named
`YYYY-MM-DD-<slug>.md`.

Plans are produced by the `superpowers:writing-plans` skill (or equivalent
per the active operating model) and must be approved before any
implementation begins. See `../../project-planning/11-execution-cycle/`
for the rules that govern plan creation and review.

The historical v0.1.0 plan lives under `../archive/superpowers/plans/`.
