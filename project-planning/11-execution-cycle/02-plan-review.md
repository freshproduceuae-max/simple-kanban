Your role is ADVISORY.

If the project folder location is not already obvious from the workspace, ask me for it first.

Before reviewing the plan:

1. Read CLAUDE.md first.
2. Resolve canonical docs paths from CLAUDE.md, including the project's operating-model record if one exists.

5. Use the active phase/workstream docs subdirectory if one exists.

## Review For

1. Alignment with governing docs
2. Completeness
3. Simplicity opportunities
4. Risk flags
5. Sequence
6. Design fidelity
