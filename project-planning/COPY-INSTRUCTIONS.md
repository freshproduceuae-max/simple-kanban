# Copy Instructions

Use this file when moving the planning skeleton into a new project.

## Copy Steps

1. Copy the entire `project-planning/` folder into the root of the new project.
2. Open the project in your AI environment.
3. Use this exact phrase:

`Refer to the project planning folder so we can begin our work.`

## Expected AI Behavior

The AI should:

1. read `START-HERE.md`
2. read `README.md`
3. ask for the operating model
4. establish canonical docs folders if needed
5. proceed through the phases in order

## Do Not Skip

- operating model selection
- readiness
- vision
- bootstrap
- PRD
- feature list
- tracking docs

Do not jump to code before the planning skeleton says it is time.
