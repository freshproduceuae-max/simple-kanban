# Gemini Design Style Prompt

Attach:

- vision document
- inspiration screenshots
- optional Aura HTML reference

Paste into `Gemini`.

```text
I'm building a software product. I've attached my vision document — it describes what this product is, who it's for, and the tone and feeling it should have.

I've also attached visual references: screenshots of sites, apps, and design moments that feel right for this product. If I've included an HTML template file, treat that as an additional reference for structure, spacing, and visual density.

I want you to think like a world-class designer. Read my vision document carefully. Let the product's purpose, audience, and emotional intent shape every design decision.

Your job is to create a complete brand identity and design system for this product.

Cover:

1. Brand Personality
2. Voice and Tone
3. Color System
4. Typography
5. Spacing and Layout
6. Elevation and Depth
7. Motion and Animation
8. Component Personality
9. Design Token Summary

Output format:

Write this as a structured markdown document. Use clear headers. Include a short rationale beneath each major section.

Do not generate any code in this step.

Be decisive. Make real choices.
```
