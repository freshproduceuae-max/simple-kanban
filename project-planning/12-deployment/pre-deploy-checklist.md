# Pre-Deploy Checklist

Confirm all of these before production deployment:

- approved PR exists
- blocking review passed
- feature tracking docs updated
- release record for this deployment has been created
- branch and commit SHA recorded
- Vercel project and target environment confirmed
- required environment variables confirmed
- migration status confirmed
- rollback target identified
- post-deploy verification checklist prepared
- previous known-good deployment identified

If any item is missing, pause deployment.
