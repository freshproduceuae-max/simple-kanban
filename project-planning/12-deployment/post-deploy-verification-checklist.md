# Post-Deploy Verification Checklist

Run and record these after deployment:

- deployed URL loads successfully
- primary user flow works
- auth flow works if applicable
- critical API or backend health checks pass
- logs show no immediate error spike
- feature-specific validation steps pass
- environment-specific integrations behave correctly
- previous data integrity assumptions still hold

Record:

- pass / fail for each item
- timestamp
- verifier
- any incident notes
